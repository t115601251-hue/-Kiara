# 自媒体创作素材库

一个面向自媒体创作者的资源导航网站，汇集 AI 工具、素材资源、设计平台、数据热点等常用网站链接，支持分类筛选和关键词搜索。

## 功能特点

- **分类导航**：按 AI 工具、效率工具、设计创作、数据热点、素材资源等分类浏览
- **关键词搜索**：实时搜索过滤，快速定位目标工具
- **响应式设计**：适配桌面端和移动端
- **暗色主题**：护眼暗色界面，适合长时间使用

## 本地预览

直接在浏览器中打开 `index.html` 即可使用。
